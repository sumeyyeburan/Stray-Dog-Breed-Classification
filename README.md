# Stray Dog Breed Classification

AI-powered stray dog breed classification system using:

- YOLOv8
- EfficientNet-B3
- Streamlit
- PyTorch

## 🚀 Live Demo

[https://stray-dog-breed-classification.streamlit.app/](https://stray-dog-breed-classification.streamlit.app/)

## Features

- Dog detection
- Breed classification
- Hybrid analysis
- Breed information
- Dataset comparison

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
